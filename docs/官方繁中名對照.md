# 官方繁中名對照（查證備用）

> 用途：配點器/流程追蹤器需要對照 Elden Ring 官方繁體中文專名時的查證備忘。
> **權威來源**：原攻略作者 **feiouex（巴哈姆特）** 的各篇一覽，與流程追蹤器同源。
> 本檔只放「名稱對照」與來源連結，不複製攻略內文。

## 查證方法（重要，吃過虧）

- **WebFetch 對 `home.gamer.com.tw` 會回 403**（巴哈擋 WebFetch 的 User-Agent），
  但 **Bash `curl` 抓得到（200）**：
  ```bash
  curl -sS "https://home.gamer.com.tw/artwork.php?sn=XXXX" -o /tmp/x.html
  # 再用 python 去標籤解析，搜尋英文名抓鄰近中文
  ```
- 判別「真的被 egress 擋」vs「只是 UA 問題」：`curl` 回 `Host not in allowlist` = 真被擋
  （如 `wiki.biligame.com`、`*.github.io`、`b23.tv→bilibili` 風控驗證碼頁）；回 200 = 只是 UA。
- **英文搜尋摘要的音譯不可信**，官方繁中一律以 feiouex 一覽為準。
